## Synchronized MapControl and PageLayoutControl application
This sample demonstrates synchronization of a MapControl and a PageLayoutControl in one application. This means that, as in ArcMap, you can switch between the two views and maintain the scale and extent, as well as add graphics to the Map view and see them in the PageLayout view.
 
For simplicity, this sample only handles one data frame; it selects the first data frame from a map document.

Start Visual Studio, open the solution, and build the project. 1. Press F5 or double-click the resulting executable to run the application. 1. Use the Open Map Document command to load a new map document to the application. 1. Zoom in and add graphic elements using the Graphics toolbar at the bottom on the application window. 1. Click the Layout tab to switch to the Layout view. The PageLayout is centered around the same point as the MapControl, and the graphic elements from the Map view are visible in the Layout view. 1. Use the Pan tool in the PageLayout view, then switch back to the Map view.
