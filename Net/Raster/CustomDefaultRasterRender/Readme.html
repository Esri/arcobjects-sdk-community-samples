Start Visual Studio, open the solution file, and compile the program. A DLL is generated and registered in the ESRI RasterRendererMakers component category. 1. Run category.exe from the ArcGIS bin directory to confirm that the custom DLL was added to that category. 1. Start ArcMap and add a 1-bit TIFF image. The image displays as a unique value renderer with red color. 1.
