## Route layer

  <div xmlns="http://www.w3.org/1999/xhtml">This sample is an application that shows how to programmatically perform route analysis. You will learn how to load stops from a feature class, set solver parameters, execute the analysis, and write the analysis layer to a layer file that can then be used by other applications. </div>

### Usage
1. Start Visual Studio, open the solution file, then build the solution.  
1. Run RouteLayer.exe.  
1. Add the layer Route.lyr that was generated by this application to view the results in ArcMap.
