## Location-allocation solver

This sample application demonstrates how to programmatically perform a Location<font xmlns="http://www.w3.org/1999/xhtml" face="Verdana">–</font>Allocation analysis. This sample shows how to load locations from a feature class, set solver parameters, execute the analysis, and obtain output results.

### Usage
1. Use Visual Studio to build and run the application.  
1. In the application, click Choose Optimal Facilities Location to perform the analysis. The solution is displayed.  
1. Optionally, change the solver settings and re-solve.
